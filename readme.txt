Running Program:
- Download Python on Unix system
- Download zip folder with contents with .txt and .py files and place in same directory
- Go into terminal and cd into the correct directory
- run command: python project1_starter.py
- Note: If you are downloading this directly from github and not the zip folder with "project1_starter.py", then run "main.py" instead of "project1_starter.py"
